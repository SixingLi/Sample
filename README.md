# SimOne 仿真平台API模块
[![Build Status](https://app.travis-ci.com/DTC8/DTC.svg?branch=master)](https://app.travis-ci.com/github/DTC8/DTC)

## 概述
SimOne 无人驾驶分布式仿真平台API模块，允许用户接入感知或决策算法，实现算法的测试或训练

## 接口

- [x] C++

- [x] Python

- [x] Simulink

## 目录
- ***Build：*** 构建目录
- ***InternalTest：*** 内部测试脚本
- ***Matlab：***Simulink接口及示例
- ***Samples：***C++及Python示例
- ***SourceCpp：***C++接口代码
- ***SourcePython：***Python接口代码
